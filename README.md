# Follow-up Comercial Soretrac Portuguesa

Aplicacao desktop para gestao de follow-up comercial.

## Inicio rapido

- **Admin:** `build_output/SoretracAdmin.exe`
- **Funcionarios:** recebem `Soretrac_Funcionarios.exe` (um unico ficheiro)

## Formatos de importacao

CSV, Excel (.xlsx), SQL (.sql), SQLite (.db) — detecao automatica.

## Desenvolvimento

```bash
pip install -r requirements.txt
python main.py
```

## Versao

3.0.0 — Soretrac Portuguesa
